## Getting Started

### Prerequisites
- Flutter SDK (3.24.0 or higher)
- Android Studio or VS Code
- Android SDK

### Installation

1. Clone the repository:
```bash
git clone https://github.com/qajohn121/christify.git
cd christify
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```

### Building APK

To build a release APK:
```bash
flutter build apk --release
```

The APK will be located at: `build/app/outputs/flutter-apk/app-release.apk`

## Automated Builds

This repository uses GitHub Actions to automatically build APK files on every push to main branch.

Download the latest APK from the [Releases](https://github.com/qajohn121/christify/releases) page.
